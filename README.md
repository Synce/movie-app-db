# Movie database app

Movie fetching app. (Recruitment Task)

## Installation

Add token.ts with your API_TOKEN from [The Movie Database](https://developers.themoviedb.org/)

Supports genre and year filtering, searching by query, and infinite scrolling. 

## Used 
* Expo
* React-Native
* Redux
* Native-Base
* React Navigation
* Fonts-awesome
* Typescript
